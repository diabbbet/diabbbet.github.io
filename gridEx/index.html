Геодецентрализированная инфраструктура GridEx

решает проблему ликвидности значимых криптовалютных активов как для обычных пользователей, так и для банковских групп

Мы представляем безопасную и высокопроизводительную инновационную инфраструктуру круглосуточно для автоматического и ручного трейдинга.

Поддерживается как высокочастотный трейдинг через отдельный API до 100 тысяч транзакций в секунду для отдельного стакана, так и торговлю посредством отдельного торгово-аналитического терминала с богатой поддержкой технического анализа ТА и возможностью написания собственных индикаторов и торговых экспертов.

Мы разрабатываем экосистему, позволяющую каждому участнику использовать криптовалютные активы и конвертировать их с минимальными задержками и максимальной скоростью.

Зарегистрированный пользователь нашей экосистемы GridEx может выбрать необходимый ему функционал, состоящий из нескольких возможных сценариев использования: через смартфон, торговый терминал, сайт биржи либо в магазинах, использующих экосистему GridEx для приема оплаты в криптовалюте.

Для обеспечения этих задач мы разработали высокопроизводительную инфраструктуру, рассчитанную на масштабирование до миллионов транзакций в секунду, центром которой является наша биржа, выступающая бассейном ликвидности для остальных сервисов.

принимают участие в опросе по выявлению наиболее востребованных коинов и токинов на криптовалютном рынке

C прямым участием выпущенных токенов GEX в экосистеме GridEx через оплату комиссий при трейдинге и возможностью линейного масштабирования инфраструктуры.
Для клиентов с более классическими требованиями к функционалу GridEx будет предоставляться возможность торговать через web-браузер и отдельное торговое приложение для смартфона.
Для пользователей с особыми требованиями к безопасности будет возможность использовать криптовалютный депозитарий с невозможностью проведения операций по причине отсутствия у инфраструктуры GridEx части приватной ключа пользователя.

What is Gridex

Рынок криптовалют стремительно развивается, увеличивая охват населения, участвующего в нем. Среднесуточные объемы торгов составляют 40 млрд. долларов.*

Для сравнения-рынок форекса за сутки в среднем имеет объем не менее 3 триллионов долларов

*согласно coinmarketcap.com от 29.01.2018

Ключевым моментом для достижения таких ежесуточных объемов «консервативных» рынков, как форекс, является массированное использование высокочастотного трейдинга других технологий автоторговли.

Это значительно увеличивает ликвидность торгуемых активов, уменьшает спреды, устраняет перекосы в кросс-курсах и запускает очередной виток повышения ликвидности.

Подобное ожидает и рынок криптовалют. На текущий момент ни одна существующая биржа не позволяет в полной мере реализовывать высокочастотный трейдинг. HFT.

GridEx - геодецентрализованная экосистема торговли криптовалютной ликвидностью

Проект GridEx состоит из нескольких этапов, реализуемых либо по очереди, либо параллельно.

Геодецентрализованный бассейн ликвидности в виде криптовалютной биржи, поддерживающей высокочастотный трейдинг, основной активный серверный кластер которой будет расположен в Швейцарии.

Архитектура биржи и её исполняемое ядро будет поддерживать обработку 100 000 транзакций в секунду для отдельного торгуемого инструмента с линейным масштабированием.

Многопользовательская инфраструктура конвертации криптовалют через смартфоны Android/IOS, с возможностью отправления криптовалют третьим лицам.

Любой пользователь сможет конвертировать криптовалюты, скачав приложение GridEx, и зарегистрировавшись в нашей инфраструктуре, в удобном ему направлении по интерактивно обновляемому курсу с указанием текущей цены и максимального объема конвертации по данной цене.

Объем рынка конвертаций криптовалют через смартфоны на конец 2018 года предполагается не менее 2 млрд. долларов в сутки.

Многопользовательская инфраструктура для продавцов любого актива, зарегистрированного в GridEx, для продажи собственных товаров за криптовалюту.

Мировой рынок услуг и товаров на текущий день превышает 10 трлн. долларов в год. Возможный рост рынка услуг и товаров, покупаемых за криптовалюту, к 2021 году составит до 2 %, что составит 200 млрд. долларов.

Комиссия нашей инфраструктуры при отправлении криптовалюты на кошелек продавца будет состоять из комиссии криптосети+1% комиссии нашей инфраструктуры.
Назад
Реализация отдельно включаемого механизма асимметричного шифрования данных пользователя с хранением приватной части ключа самим пользователем.

Без данного ключа невозможно получить доступ к информации о его счетах и проводимых операциях (даже в случае полной компрометации одновременно и активного серверного кластера и инфраструктуры, работающей с ключами шифрования).

Реализация инфраструктуры GridEx для децентрализованного хранения приватных ключей криптовалютных активов в зашифрованном виде, переданных на биржу для торговли.

Данная инфраструктура полностью защищает пользователя от несанкционированного доступа к его не торгуемым активам любых злоумышленников, не позволяя свободно бирже и третьим лицам распоряжаться его активами (т.к. биржа не сохраняет пользовательскую часть приватного ключа).

Страхование со взносами и страховой выплатой через смарт-контракты. Клиенты биржи смогут застраховать свои риски, произошедшие в границах инфраструктуры биржи.
Создание нескольких одновременно работающих криптовалютных бирж в основных финансовых центрах планеты
Отдельное приложение в виде торгового терминала с расширенным набором инструментов технического анализа

(включая графики Kagi, Renko, P&F, Three Line Break, Candles) с произвольным таймфреймом от 1 до 10080, с возможностью написания собственных индикаторов и торговых стратегий на любом языке под .NET CLR, с возможностью тестирования торговых стратегий на исторических данных и оптимизации их параметров.

Реализация криптовалюты GridEx, первая реализация на F#, с поддержкой Тьюринг неполных смарт-контрактов для исполнения страховых обязательств биржи, и эффективного хранения зашифрованных половинок приватного ключа пользователя (реализованный STAGE5), с миграцией токенов GEX на данную криптовалюту по желанию пользователя с последующим сжиганием его токенов GEX в блокчейне Ethereum через процедуру переноса токенов через инфраструктуру биржи GridEx.

Токен GEX

предполагает использование в следующих сценариях

1
Оплата комиссий экосистемы GridEx токенами GEX при совершении торговых сделок в любом направлении со скидкой в 50% от той комиссии в течении первого года работы платформы, что пришлось бы платить при совершении сделки без использования оплаты токенами GEX.
2
Владельцы токенов принимают участие в опросе по выявлению наиболее востребованных коинов и токенов на криптовалютном рынке, т.к. биржа последовательно будет увеличивать количество оборудования, а значит, мы вынуждены создавать приоритет добавления торгуемых инструментов на биржу. В итоге из топ 5 инструментов, за который будет внесено наибольшее количество токенов, биржей будет выбран один и добавлен для листинга на бирже. Мониторинг будет проходить с определенной периодичностью, по мере масштабирования всей архитектуры биржи
3
Токен GEX будет торговаться на самой бирже GridEx, поэтому все желающие купить либо продать GEX всегда будут иметь возможность это сделать в рамках объема торгового стакана GEX на нашей бирже.
