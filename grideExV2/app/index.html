Q4 2018

stage 1

Запуск криптовалютной биржи, поддерживающей HFT (High Frequency Trading), основной активный серверный кластер с торгуемыми инструментами будет расположен в Швейцарии. В случае получения разрешения регуляторов соответствующих стран - будут открыты дополнительные кластеры для торговли в данных странах.

Торгуемые инструменты при старте биржи GEX/BTC, GEX/ETH, GEX/LTC, GEX/XTZ, XTZ/BTC, ETH/BTC, LTC/BTC.

Для торговли доступен веб-клиент и API для алготрейдинга. В дальнейшем будут добавляться криптовалюты с высокой ликвидностью и высоким рейтингом при голосовании токенами GEX.

Q1 2019

stage 2

Отдельное приложение в виде торгового терминала с расширенным набором инструментов технического анализа и с поддержкой пользовательских плагинов технического анализа и автотрейдинга.

Q2 2019

stage 3

страхование рисков простоя биржи через смарт-контракт Ethereum c выплатой страховки либо в ETH, либо в токенах GEX

Q3 2019

stage 4

Цифровой депозитарий для пользователей с особыми требованиями к безопасности.

Q4 2019

stage 5

реализация высокопроизводительного блокчейна CEX, reference реализация на F#, с поддержкой смарт-контрактов без Тьюринг полноты, для исполнения страховых обязательств биржи и других задач инфрастуктуры GridEx с последующим переносом токенов GEX с блокчейна Ethereum на блокчейн GEX

2020

stage 6

запуск дополнительных дата-центров биржи в разных регионах планеты, имеющих разные стаканы для торговли, но общие балансы пользователей
